# SAGE-UNet Project Website

This repository contains the source code for the SAGE-UNet project website, hosted on GitHub Pages.

## How to Run Locally

To preview the website on your local machine:

1.  Open a terminal in this directory.
2.  Run a simple Python HTTP server:
    ```bash
    python3 -m http.server 8000
    ```
3.  Open your browser and navigate to:
    [http://localhost:8000](http://localhost:8000)

## How to Deploy

This website is hosted using GitHub Pages. To deploy changes:

1.  Commit your changes:
    ```bash
    git add .
    git commit -m "Update website content"
    ```
2.  Push to the repository:
    ```bash
    git push origin main
    ```
3.  The site will be live at `https://Lusic12.github.io` (it may take a few minutes to update).
